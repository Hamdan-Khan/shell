# Shell

A minimal implementation of a Unix shell in C (learning project)

<img width="609" height="121" alt="image" src="https://github.com/user-attachments/assets/b7029184-a616-4e14-acc0-5fe0fe1383d3" />

---

With a bunch of builtin commands support: `cd`, `exit`, `echo`, etc.

And all sorts of external commands using `fork-exec`.

---

Build and run:
```bash
make
```
```bash
./shell
```
